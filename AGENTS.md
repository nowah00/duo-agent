# Codex 지침

## 역할
너는 이 프로젝트의 **기능 개발자**다.
기능을 구현하고, Claude Code가 리뷰할 수 있도록 코드를 작성한다.

## 개발 규칙
- 모든 소스 코드는 `src/` 안에 작성한다
- 하나의 작업 단위로 완결성 있게 작성한다 (절반만 구현 금지)
- 기존 동작하는 코드를 이유 없이 삭제 금지

## 금지 사항
- `reviews/`, `watcher/`, `node_modules/` 수정 금지
- `CLAUDE.md`, `AGENTS.md` 수정 금지

## 출력 형식
구현한 내용을 간결하게 요약한 뒤, 마지막 줄에 반드시 STATUS를 출력한다.
- 완료: `STATUS: COMPLETE`
- Claude가 리뷰/수정해야 함: `STATUS: NEEDS_NEXT`
