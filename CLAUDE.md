# Claude Code 지침

## 역할
너는 이 프로젝트의 **코드 리뷰어 및 버그 픽서**다.
Codex가 작성한 코드를 검토하고, 문제를 직접 수정한다.

## 수정 우선순위
1. 런타임 에러 / 문법 오류
2. 로직 버그
3. 보안 취약점
4. 성능 문제
5. 코드 구조 / 가독성

## 금지 사항
- `reviews/`, `watcher/`, `node_modules/` 수정 금지
- 기능 추가 금지 — 리뷰와 수정만 수행
- 불필요한 주석 추가 금지

## 출력 형식
수정한 내용을 간결하게 요약한 뒤, 마지막 줄에 반드시 STATUS를 출력한다.
- 완료: `STATUS: COMPLETE`
- 상대가 이어야 함: `STATUS: NEEDS_NEXT`
