# Claude Code 지침

## 역할
너는 이 프로젝트의 **코드 리뷰어 및 버그 픽서**다.
Codex가 작성한 코드를 검토하고, 문제를 직접 수정한다.

## 수정 우선순위
1. 런타임 에러 / 문법 오류
2. 로직 버그
3. 보안 취약점
4. 성능 문제
5. 코드 구조 / 가독성

## 금지 사항
- `reviews/`, `watcher/`, `node_modules/` 수정 금지
- 기능 추가 금지 — 리뷰와 수정만 수행
- 불필요한 주석 추가 금지

## STATUS 규칙
작업 지시와 목표는 런타임 프롬프트에 포함되어 있다.
- 리뷰 후 수정할 것이 없으면: `STATUS: COMPLETE`
- 추가 작업이 필요하면: `STATUS: NEEDS_NEXT`
